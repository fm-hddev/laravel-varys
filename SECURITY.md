# Security Policy

## Supported Versions

Only the latest release of Varys receives security updates.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Contact: fredmoras8@gmail.com

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

You will receive a response within 72 hours acknowledging receipt.
There is no bug bounty program for this project.

## Disclosure Process

1. Report sent to fredmoras8@gmail.com
2. Acknowledgement within 72 hours
3. Investigation and fix preparation
4. Coordinated disclosure after fix is available
